# vue-lottie-player Vue example

Vue 3 + Vite example app for the local `vue-lottie-player` source.

## Project setup

```bash
npm install
```

## Development

```bash
npm run dev
```

The Vite config aliases `vue-lottie-player` to `../../src/index.js` so changes
are picked up directly from source.

## Production build

```bash
npm run build
```
