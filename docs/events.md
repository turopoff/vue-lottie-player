# Events

- `ready`: Emitted once per load with the created `lottie-web` animation instance.
- `error`: Emitted when the source props are invalid or the animation fails to load.

## Type Exports

- `LottieSource`
- `LottieUrlSource`
- `LottieDataSource`
- `VueLottiePlayerProps`
- `VueLottiePlayerInstance`
